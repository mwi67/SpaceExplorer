    1. Ensure Java 11 is installed on the machine
    2. Open the zip folder in a terminal window
    3. Run "java -jar mwi67_amo174_SpaceExplorer.jar"
In Eclipse
    4. Go to File ->  Click Import -> Select ”Existing Projects into Workspace”(In General Folder) 
Click Next -> in Root  directory Browse  for the file  (src  is in the folder code)->  Click Finish
    5. Select package game -> click on Game.java and hit run.
    6.  If package testing has red line on  “@Test”, hover over it -> click on JUnit5
Then right Click on package testing and Select Run As -> JUnit Test.
